# 프로필 이미지 및 토스트 닉네임 텍스트 컬러 규칙 v0.2 (정본)

## 핵심 요점
- **프로필 이미지**: 닉네임 첫 글자 원형 이미지(Nametag-Chip). 6컬러 × 2타입 = 12종, 계정당 1종 랜덤.
- **컬러 매핑**(토스트 닉네임 텍스트 + Grouptag-Chip Timestamp, 타입별 강제):
  연핑크(1,2)→Cherry-200 / 진핑크(3,4)→Cherry-300 / 체리(5,6)→Cherry-400 / 그레이(7,8)→White / 멜론(9,10)→Melon / 푸딩(11,12)→Pudding.
- **타입 고정**: 계정 생성 시 1회 랜덤·불변, 동일 닉네임·다른 계정 → 다른 타입 가능, 모든 화면·재접속·그룹 변경에도 유지.
- 개인정보: 작성자·예시 실명 마스킹(더미 `홍씨` 닉네임 두 계정).

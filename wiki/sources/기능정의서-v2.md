# 기능정의서_MVP v2 (2차)

- **원본**: `raw/기능정의서-v2.md`
- **날짜**: 2026-05-30
- **도메인**: product

## 핵심 요점
- [[협업-캔버스-앱]]의 2차 화면 단위 기능정의. 화면 33개.
- [[화면-ID-체계]] 확립: `A-`(Auth) `G-`(Group) `C-`(Canvas) `S-`(Sidebar) `SY-`(System).
- 핵심 흐름: 로그인 → 진입경로 선택(참여/생성) → [[그룹]] → 오늘자 캔버스(C-001) → 사진 촬영/갤러리 → [[누끼-따기]] → 편집 → 배치.
- [[캔버스-마감-스케줄]] 최초 정의: 매일 **00시** 캔버스 마감·갤러리 저장(SY-001), 신규 캔버스 자동 생성(SY-003).
- 그룹 최대 인원 **12명** 고정, MVP에서 그룹명/인원 추후 변경 불가.

## 이 버전의 특징
- 편집 허브 **C-002** 참조가 다수 존재하나 표에 C-002 행 자체가 없음(정의 공백).
- C-301 = "캔버스 배경 변경", C-305 = "수정 완료".

# 프로필 이미지 및 그룹칩 내 타임스탬프 컬러 규칙 v0.1

- **원본**: `raw/S-101-프로필-이미지-및-그룹칩-내-타임스탬프-컬러-규칙-v0.1.md`

- **도메인**: product
- **적용 범위**: 1. Nametag-Chip / 2. Grouptag-Chip Timestamp
- **성격**: [[S-101-프로필-닉네임-컬러-규칙-v0.3]]에서 **분리된** Grouptag-Chip Timestamp 컬러 규칙의 독립 문서.
- **연관**: [[nametag-chip]], [[G-001-무한파르페-간격-정책-v0.2]](Grouptag-Chip이 노출되는 G-001 토핑 규격)

## 핵심 요점
- **프로필 이미지**: 닉네임 첫 글자 원형 이미지(Nametag-Chip), 6컬러 × 2타입 = 12종, 계정당 1종 랜덤. (닉네임·토스트 컬러 규칙과 동일 구성.)
- **그룹칩 Timestamp 매핑**: G-001에서 그룹 대표 토핑과 함께 노출되는 Grouptag-Chip 속 **Timestamp 텍스트** 컬러는, 해당 그룹에서 **마지막으로 변화를 가한**(토핑 추가·토핑 편집·배경 추가·배경 변경) 유저의 Nametag-Chip 타입에 강제 매핑:

| 컬러 (타입) | Timestamp 텍스트 컬러 |
|---|---|
| 연핑크 (Type 1, 2) | Cherry-100 |
| 진핑크 (Type 3, 4) | Cherry-200 |
| 체리 (Type 5, 6) | Cherry-300 |
| 그레이 (Type 7, 8) | White |
| 멜론 (Type 9, 10) | Melon |
| 푸딩 (Type 11, 12) | Pudding |

- **토스트 닉네임 매핑과의 관계**: 서로 **다른 UI 요소용 별개 매핑**(상충 아님). 토스트 닉네임([[S-101-프로필-닉네임-컬러-규칙-v0.3]])은 Cherry-200/300/400, 그룹칩 Timestamp는 Cherry-100/200/300. 그레이/멜론/푸딩은 두 매핑 동일.

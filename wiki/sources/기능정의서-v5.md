## 핵심 요점
- [[협업-캔버스-앱]] 5차 기능정의. 실제 개발 착수 단계 — 담당자 배정 + 에러/확인 서브 화면 보강.
- 페이지 비고: 누락 화면명 추가, 기존 A-003(그룹추가 분기) 삭제, 담당자 정의, 비고 컬럼 삭제.

## v4 → v5 변경
- **담당자 컬럼 추가**: 디자인/iOS/AOS/Spring. 실명은 역할 코드(DSN-*/iOS-*/AOS-*)로 마스킹, 매핑은 `wiki/personal-private/담당자-매핑.md`(비공개).
- **A-003 교체**: "진입 경로 선택(그룹 참여/생성 분기)" 삭제 → A-003 = **"약관 동의"**.
- **서브 화면 신설**: C-101-confirm/-error, C-102-confirm/-error, C-103-loading, C-103-select(다중 누끼 검출 시 대상 선택).
- 편집 화면 재번호: C-104 "수동 누끼 편집", C-105 "수동 편집(테두리)", C-106 "캔버스에 배치"(배치+확정 통합).
- C-101/102 "커스텀 카메라/갤러리 호출"로 명시(네이티브 → 커스텀).
- A-002 "프로필 이미지 넣을지 말지" → **"안넣음"** 확정.
- [[캔버스-마감-스케줄]]: SY-001-closed(마감 갤러리 저장), SY-001-new(신규 캔버스)로 재명명.

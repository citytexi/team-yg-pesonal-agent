## 핵심 요점
- **측정 기준**: 갭은 **Topping 180×180 프레임** 기준으로 측정. 회전된 Img가 프레임 밖으로 삐져나와도 갭은 프레임 기준(회전 Img 아님).
- 갭 규칙 (2-1-2-1 반복):

| 구간 | 방향 | 갭 |
|---|---|---|
| 체리 ↔ 첫 Type 1 | 세로 | 12px |
| Type 1 ↔ Type 2-Left | 세로 | 12px |
| Type 1 ↔ Type 2-Right | 세로 | 24px |
| Type 2-Left ↔ Type 2-Right | 가로 | 0px |
| Type 2-Right ↔ Type 1 | 세로 | 12px |

- 끝 처리: 마지막 2-Left ↔ 접시 세로 **12px**, 마지막 2-Right ↔ 접시 세로 **0px**.
- 레이아웃: 체리 자리(최상단) → 토핑 스태거 → 파르페 접시 자리(최하단).

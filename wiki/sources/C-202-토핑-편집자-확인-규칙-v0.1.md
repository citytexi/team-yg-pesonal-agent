# 토핑 편집자 확인 규칙 (C-202) v0.1

- **원본**: `raw/C-202-토핑-편집자-확인-규칙-v0.1.md`
- **날짜**: 2026-07-14
- **버전**: v0.1 (초안)
- **작성자**: DSN-B (실명 마스킹, [[협업-캔버스-앱]])

- **적용 화면**: C-202 (작업한 그룹원 닉네임 노출 · Canvas) ([[기능정의서-v5]])

## 핵심 요점
- 기능정의서 표 1행이던 **C-202 "작업한 그룹원 닉네임 노출"** 이 독립 정책서로 확장 — 상호작용을 **Spotlight 상태 + 작성자 Toast**로 정의.
- **타인 토핑 탭 → Spotlight**: 선택 토핑만 강조, 나머지 캔버스 영역 Dim. 진입과 동시에 작성자 Toast 1회 노출.
- **본인 토핑 탭 → 본 정책 미적용, C-305 토핑 편집 진입.**
- **상태**: Default(모든 토핑 동일 레벨) ↔ Spotlighted(1개만 강조 + Dim).
- **상태 전이**:
  - Default → Spotlighted: 타인 토핑 탭.
  - Spotlighted → Default: Spotlight 토핑 제외 영역(=Dim 포함) 탭.
  - Spotlighted → Spotlighted: **직접 전환 불가**. 다른 토핑 선택하려면 반드시 Default 경유. (원본에 "안 거쳐도 되게 개발 가능하면 환영" 취지 디자인 주석 있음.)
- **Spotlight 렌더링**:
  - Dim 레이어 = **Transparency/Black-50**, 선택 토핑 제외 전체 영역 적용.
  - z-index 우선순위: **Spotlight 토핑 > Dim 레이어 > 나머지 토핑 > 배경**. Spotlight 토핑만 최상위로 이동, 나머지는 기존 z 유지.
- **Toast 정책**: Spotlight 진입 시 1회. 문구 `{닉네임} 님이 {상대시간} 전에 쌓았어요`. 노출/소멸은 [[toast|Toast 공통 규칙]] 준수.
- **엣지 케이스**:
  - 탈퇴/그룹 탈퇴 유저 토핑도 Spotlight 정상 동작, Toast만 **'알 수 없음'** 표기(예: `알 수 없는 사용자가 59분 전에 쌓았어요`).
  - 앱 백그라운드→복귀: Spotlight 해제, Default 복귀.
  - Pull-to-Refresh: 먼저 Default 복귀 후 새로고침.

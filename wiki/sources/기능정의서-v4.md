## v3 → v4 변경
- **명칭 [[누끼-따기|누끼]] → [[토핑]]**: C-305 "토핑 편집", C-306 "토핑 테두리 편집", C-202 "작업한 친구"→"작업한 그룹원 닉네임 노출".
- **그룹 내 닉네임 입력 추가**: A-004(참여)·A-005(생성) 사용자 액션에 닉네임 입력 단계.
- **[[캔버스-마감-스케줄]] 시각 변경 00시 → 03시**: SY-001/SY-003 매일 03:00(KST).
- **갤러리 시간 창 규칙**: C-102는 당일 03시~익일 03시 사진만 로드(하루 경계 = 03시). 예: 6/21 캔버스 = 6/21 03시~6/22 03시.
- S-004 "웹뷰 vs 네이티브 결정 필요" → **"노션 웹뷰"** 확정.
- 그룹장 기능 삭제 완전 반영(S-103 분기/비고 비움).

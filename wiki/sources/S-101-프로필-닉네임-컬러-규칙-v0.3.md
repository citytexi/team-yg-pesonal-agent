# 프로필 이미지 및 토스트 닉네임 텍스트 컬러 규칙 v0.3 (정본)

- **원본**: `raw/S-101-프로필-닉네임-컬러-규칙-v0.3.md`
- **날짜**: 2026-07-17 (작성자 DSN-B)
- **도메인**: product
- **적용 범위**: 1. Nametag-Chip / 2. Toast > Username. (v0.2의 3. Grouptag-Chip은 **분리**되어 이 규칙 범위에서 제외)
- **비고**: v0.2([[S-101-프로필-닉네임-컬러-규칙-v0.2]])를 **대체**하는 정본. v0.2 대비 Grouptag-Chip 분리(별도 문서 [[S-101-프로필-이미지-및-그룹칩-내-타임스탬프-컬러-규칙-v0.1]]로 이관) → 규칙 범위가 Nametag-Chip·Toast Username 2개로 회귀. Grouptag-Chip Timestamp는 **별개 매핑**(Cherry 단계 다름)이라 이 규칙 표와 값이 같지 않음.

## 핵심 요점
- **프로필 이미지**: 닉네임 첫 글자 원형 이미지(Nametag-Chip). 6컬러 × 2타입 = 12종, 계정당 1종 랜덤.
- **타입 고정**: 계정 생성 시 1회 랜덤·불변, 동일 닉네임·다른 계정 → 다른 타입 가능, 모든 화면·재접속·그룹 변경에도 유지.
- **컬러 매핑**(토스트 닉네임 텍스트, 타입별 강제):
  연핑크(1,2)→Cherry-200 / 진핑크(3,4)→Cherry-300 / 체리(5,6)→Cherry-400 / 그레이(7,8)→White / 멜론(9,10)→Melon / 푸딩(11,12)→Pudding.
- 개인정보: 작성자·예시 실명 마스킹(더미 `홍씨` 닉네임 두 계정).

## v0.2 대비 변경
- **3. Grouptag-Chip 분리** — 매핑 대상이 v0.2의 "토스트 닉네임 + Grouptag-Chip Timestamp"에서 **토스트 닉네임 텍스트**만으로 축소. Grouptag-Chip Timestamp는 별도 문서([[S-101-프로필-이미지-및-그룹칩-내-타임스탬프-컬러-규칙-v0.1]])로 분리 — **별개 매핑**(Cherry 100/200/300, 이 규칙은 200/300/400).

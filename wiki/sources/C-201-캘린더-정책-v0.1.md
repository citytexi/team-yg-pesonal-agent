# 캘린더 컴포넌트 정의 (C-201) v0.1

- **원본**: `raw/C-201-캘린더-정책-v0.1.md`
- **날짜**: 2026-07-12 (작성자 DSN-B)
- **도메인**: product
- **적용 화면**: C-201 (날짜 선택 및 이동 · 캘린더) ([[기능정의서-v5]])
- **구성**: Button-Date / List-Date / Chip-Indicator

## 핵심 요점
- **Button-Date**(일자 버튼) 4상태: Default(그 외) / Selected(선택 날짜) / Today(오늘) / Disabled(캔버스를 볼 수 없는 미래 날짜). 우선순위 = **Selected + Today 겹치면 Selected로 노출**.
- **Chip-Indicator**(토핑 인디케이터): Upload/True = 해당 날짜 토핑 1개 이상, Upload/False = 0개.
- **List-Date**(캘린더 셀) = Button-Date + Chip-Indicator. True/False 기준 동일. **예외: Button-Date가 Disabled면 항상 False**.
- Disabled = 미래 날짜(캔버스 열람 불가) → [[캔버스-마감-스케줄]]의 날짜별 캔버스와 연동.

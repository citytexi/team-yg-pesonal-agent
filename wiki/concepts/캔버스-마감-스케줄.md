# 캔버스 마감 스케줄

[[협업-캔버스-앱]]은 그룹마다 **하루 1장**의 캔버스를 쓴다. 자정/새벽에 오늘자 캔버스를 마감하고 새 캔버스를 자동 생성하는 시스템 배치(SY-*).

## 동작
- **마감 + 갤러리 저장** (SY-001 / v5 `SY-001-closed`): 오늘자 캔버스 마감 → 완료 캔버스 이미지 추출 → 그룹 보존 + 사용자 갤러리 저장 옵션. 마감 시에만 저장 버튼 노출. 완료된 캔버스만 추출 가능.
- **신규 캔버스 자동 생성** (SY-003 / v5 `SY-001-new`): 그룹별 오늘자 빈 캔버스 생성. **16:9 비율**. 첫 진입자 빈 상태 트리거 시 fallback 생성.

## 실행 시각 변경
- [[기능정의서-v2]]·[[기능정의서-v3]]: 매일 **00:00**.
- [[기능정의서-v4]]부터 매일 **03:00 (KST)** 로 변경. 서버 기준 KST 고정.
- 연동: [[기능정의서-v4]] C-102 갤러리 로드 범위도 "당일 03시~익일 03시"로 하루 경계를 03시에 맞춤.

## 미결
- 마감 시점에 작업 중인(미확정) 객체 처리 정책 미정 ([[open-questions]]).
- 과거 캔버스 조회 가능 기간(예: 30일) 미정.

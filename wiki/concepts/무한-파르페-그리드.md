# 무한 파르페 그리드

[[협업-캔버스-앱]]의 그룹 목록 메인 화면(G-001) 레이아웃. 그룹 하나가 파르페 위 **[[토핑]]** 한 개로 표현되고, 그룹이 늘수록 파르페가 아래로 무한히 길어진다. 디저트(파르페) 메타포를 화면 전체에 확장한 것.

## 배치 패턴 (v0.2 현행 — 지그재그, [[G-001-무한파르페-간격-정책-v0.2]])
- 토핑 **행당 1개**, 좌/우를 번갈아(지그재그) 배치. 위→아래 좌 → 우 → 좌 → 우 무한 반복. 첫 토핑은 Left/Right 고정 시작.
- **인덱스 규칙** (0부터, 2개 단위 반복):

| index % 2 | 사이드 |
|---|---|
| 0 | Left |
| 1 | Right |

- **저개수**: `N=0` → 토핑 없음(툴팁 노출). `N=1~3` → 스크롤·파르페 늘어남 없음. **N=4부터** 크림 개수 증가.
- 레이아웃: 체리 자리(최상단, 고정) → 토핑 지그재그 → 파르페 접시 자리(최하단, 고정). 크림은 토핑 총량만큼 세로로 늘어남.
- 각 [[토핑]]은 6타입(`Left-1/2/3`·`Right-1/2/3`)의 Translate·Rotate·Chip Offset 고정 상수로 살짝 기울여 얹는다. 상세 규격 → [[G-001-무한파르페-간격-정책-v0.2]] / [[토핑]].

> 🔁 **구 모델(대체됨)**: v0.1/[[G-001-무한파르페-정책설계]]의 `2-1-2-1` 스태거(index%3, 2열/1열)·프레임 180·3타입(1/2-Left/2-Right)은 v0.2 지그재그로 대체됨. 아래는 이력.

## 측정 기준 (v0.2)
- 모든 간격은 **토핑 프레임 160×160** 기준. Img 회전·이동으로 프레임 밖으로 삐져나가는 픽셀은 측정 대상 아님.
- 세부 갭은 개별 수치 표 대신 타입별 Translate/Rotate/Chip Offset 상수로 표현 → [[토핑]] 규격 표.

<details><summary>구 갭 표 (v0.1, 대체됨 — 180 프레임·2-1-2-1)</summary>

| 구간 | 방향 | 갭 |
|---|---|---|
| 체리 ↔ 첫 Type 1 | 세로 | 12px |
| Type 1 ↔ Type 2-Left | 세로 | 12px |
| Type 1 ↔ Type 2-Right | 세로 | 24px |
| Type 2-Left ↔ Type 2-Right | 가로 | 0px |
| Type 2-Right ↔ Type 1 | 세로 | 12px |

- 끝 처리: 마지막 2-Left ↔ 접시 세로 12px, 마지막 2-Right ↔ 접시 세로 0px.
</details>

## 파르페 요소 (z-index)
아래→위 순서로 쌓임.

| 요소 | z-index | 역할 |
|---|---|---|
| 접시 | 1 | 기능 없음(장식) |
| 크림 | 2 | 배경 패딩. 가로 길이 1안(그리드 전체)/2안(풀행 중앙) 미정 |
| 체리 | 3 | 차후 기능, TBD |
| [[토핑]] / 라벨 | 4 | 그룹 대표 프리뷰 + 이름·시간 칩 |

## 라벨 정책
- 이름 칩: 한 줄 고정, 토핑 우측 하단. 폭은 내용 기준(토핑 폭 비종속).
  - 이름 **최대 6자(한글)**, 7자+ → **앞 5자 + `…`** (예: `우리동네산책모임` → `우리동네산…`).
- **상대 시간**(대표 토핑 마지막 활동 시각 기준, 항상 전체 노출):

| 경과 | 표기 |
|---|---|
| 60초 미만 | 방금 전 |
| 1~59분 | N분 전 |
| 1~23시간 | N시간 전 |
| 1~6일 | N일 전 |
| 7일 이상 | 오래 전 |

- **NEW 기준**: C-001 진입해야 읽음 처리(화면 노출만으로 해제 안 됨). 서버 unread 없을 때 폴백 = `lastActivityAt` 24시간 이내면 NEW.

## 상호작용
- 대표 토핑 클릭 → 개별 토핑을 잡고 있는 **C-001**(그룹 캔버스 메인)로 이동.
- 그룹 나가기 → 해당 토핑 제거, 뒤 토핑이 가입순대로 한 칸씩 당겨짐 → N−1로 패턴 전체 재계산(마지막 행/전체 행수 변동). 크림 수축·접시 상승. 저장·재진입 시 반영(실시간 아님).

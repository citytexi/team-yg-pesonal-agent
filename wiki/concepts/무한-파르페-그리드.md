# 무한 파르페 그리드

[[협업-캔버스-앱]]의 그룹 목록 메인 화면(G-001) 레이아웃. 그룹 하나가 파르페 위 **[[토핑]]** 한 개로 표현되고, 그룹이 늘수록 파르페가 아래로 무한히 길어진다. 디저트(파르페) 메타포를 화면 전체에 확장한 것.

## 2-1 스태거 배치 패턴
- 리스트를 `2 - 1 - 2 - 1 …` 로 반복 배치. 한 행 2개, 다음 행 1개(full-width) 교대.
- 2열 행은 왼→오 순 배치.
- **인덱스 규칙** (0부터, 3개 단위 반복):

| index % 3 | 컬럼 | span |
|---|---|---|
| 0 | 2열 좌측 | 1 |
| 1 | 2열 우측 | 1 |
| 2 | 1열 | 2 |

- **끝 처리**: 마지막 2열 자리에 1개만 남으면 좌측 1칸만 채우고 우측 비움(좌측 정렬). 데이터 수에 따라 2열로 끝날 수 있음.
- **저개수**: `N=0` → 토핑 없음, 크림 최소 + 접시, 상단 42% 지점에 빈 상태 안내("첫 그룹을 만들어 / 파르페를 쌓아보세요"). `N=1` → 가운데 1개.
- 2열/1열 네이밍은 논의 필요.

## 파르페 요소 (z-index)
아래→위 순서로 쌓임.

| 요소 | z-index | 역할 |
|---|---|---|
| 접시 | 1 | 기능 없음(장식) |
| 크림 | 2 | 배경 패딩. 가로 길이 1안(그리드 전체)/2안(풀행 중앙) 미정 |
| 체리 | 3 | 차후 기능, TBD |
| [[토핑]] / 라벨 | 4 | 그룹 대표 프리뷰 + 이름·시간 칩 |

## 라벨 정책
- 이름 칩: 한 줄 고정, 토핑 우측 하단. 폭은 내용 기준(토핑 폭 비종속).
  - 이름 **최대 6자(한글)**, 7자+ → **앞 5자 + `…`** (예: `우리동네산책모임` → `우리동네산…`).
- **상대 시간**(대표 토핑 마지막 활동 시각 기준, 항상 전체 노출):

| 경과 | 표기 |
|---|---|
| 60초 미만 | 방금 전 |
| 1~59분 | N분 전 |
| 1~23시간 | N시간 전 |
| 1~6일 | N일 전 |
| 7일 이상 | 오래 전 |

- **NEW 기준**: C-001 진입해야 읽음 처리(화면 노출만으로 해제 안 됨). 서버 unread 없을 때 폴백 = `lastActivityAt` 24시간 이내면 NEW.

## 상호작용
- 대표 토핑 클릭 → 개별 토핑을 잡고 있는 **C-001**(그룹 캔버스 메인)로 이동.
- 그룹 나가기 → 해당 토핑 제거, 뒤 토핑이 가입순대로 한 칸씩 당겨짐 → N−1로 패턴 전체 재계산(마지막 행/전체 행수 변동). 크림 수축·접시 상승. 저장·재진입 시 반영(실시간 아님).

# Nametag-Chip (프로필·닉네임 컬러 규칙)

[[협업-캔버스-앱]]에서 유저를 표현하는 **닉네임 첫 글자가 들어간 원형 프로필 이미지**. 프로필·토스트 닉네임·[[토핑]] Grouptag-Chip Timestamp의 텍스트 컬러를 일관되게 결정하는 기준. S-101(그룹 사이드 메뉴)에서 다른 유저를 볼 때 노출.

## 구성
- **6개 컬러**(연핑크·진핑크·체리·그레이·멜론·푸딩) × **각 2타입** = **총 12종**.
- 계정당 **12종 중 1종 랜덤** 배정.

## 컬러 매핑 (토스트 닉네임 텍스트 + Grouptag-Chip Timestamp)
공용 캔버스 액션 발생 시 동접 그룹원에게 뜨는 토스트의 **닉네임 텍스트**, 그리고 Grouptag-Chip의 **Timestamp 텍스트** 컬러는 유저 Nametag-Chip 타입에 **강제 매핑**된다.

| 컬러 (타입) | 텍스트 컬러 |
|---|---|
| 연핑크 (Type 1, 2) | Cherry-200 |
| 진핑크 (Type 3, 4) | Cherry-300 |
| 체리 (Type 5, 6) | Cherry-400 |
| 그레이 (Type 7, 8) | White |
| 멜론 (Type 9, 10) | Melon |
| 푸딩 (Type 11, 12) | Pudding |

## 타입 고정 정책 (v0.2 추가)
- 타입은 **계정 생성 시 1회 랜덤 배정**, 이후 변경 없음.
- **동일 닉네임이라도 계정이 다르면 서로 다른 타입** 가능.
- 배정 타입은 모든 화면에서 동일 유지, 앱 재접속·그룹 변경에도 불변. 별도 타입 변경 기능 없는 한 계정 유지 동안 동일 타입.
- 예: 계정 A(닉네임 `홍씨`) → Type 1 고정, 계정 B(닉네임 `홍씨`) → Type 8 고정. 닉네임 같아도 계정 다르면 컬러 다름.

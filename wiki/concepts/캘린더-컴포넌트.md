# 캘린더 컴포넌트 (C-201)

[[협업-캔버스-앱]]에서 날짜를 선택·이동할 때(C-201) 노출되는 캘린더의 컴포넌트 정의. 일자 버튼(Button-Date) + 토핑 업로드 표시(Chip-Indicator)를 합친 셀(List-Date)로 구성.

## Button-Date (일자 버튼) — 4상태
| 상태 | 정의 |
|---|---|
| Default | Selected·Today·Disabled가 아닌 모든 경우 |
| Selected | 사용자가 선택한 날짜 |
| Today | 오늘 날짜 |
| Disabled | 캔버스를 볼 수 없는 **미래의 날짜** |

- **우선순위**: Selected와 Today가 겹치면 **Selected로 노출**.

## Chip-Indicator (토핑 인디케이터)
- **Upload/True**: 해당 날짜 업로드 [[토핑]] **1개 이상**.
- **Upload/False**: 해당 날짜 토핑 **0개**.

## List-Date (캘린더 셀)
- **Button-Date + Chip-Indicator** 조합. True/False 판정 기준은 Chip-Indicator와 동일.
- **예외**: Button-Date가 **Disabled**면 Chip-Indicator는 **항상 False**.
- 시각 케이스: Default/선택/오늘 × 토핑 O/X, 그리고 미래 날짜(Disabled → 토핑 X 고정).

## 연관
- [[캔버스-마감-스케줄]] — 날짜별 캔버스 존재·미래 날짜 열람 불가(Disabled)와 연동.
- [[토핑]] — Chip-Indicator의 업로드 판정 대상.

# 캔버스 반응형 레이아웃 (C-001)

[[협업-캔버스-앱]]의 그룹 캔버스 메인 화면(C-001) 렌더링 규칙. 화면 크기가 달라져도 캔버스 비율·여백을 일관되게 유지하는 반응형 정책과, 배경 위에 항상 얹는 Background Blur 레이어 정책. **세로 모드 전용**(태블릿·가로모드 미고려).

## 탑바 / 탭바 / 날짜 라벨
- 탑바(상단)·탭바(하단): **플로팅 버튼 형태**로 화면 상/하단 고정, 좌우 패딩 **20px**.
- 날짜 라벨(`May 20 (Wed)`): 크기 고정(화면 크기 무관, 스케일 X), 위치는 캔버스 좌상단 고정.

## 캔버스 크기·위치 우선순위
1. **16:9 비율 무조건 유지** (최상위).
2. 탑바–탭바 사이 상하 **최소 gap 12px** 보장.
3. 좌우 패딩 **20px** 유지 (가능한 한도 내).

## 배치
- 탑바–탭바 사이 **세로 중앙 정렬**, 상단·하단 갭 항상 동일(최소 12px).
- **평소(세로 공간 충분)**: 좌우 패딩 20px 유지 → 캔버스 너비 = 탑바/탭바와 동일.
- **예외(가로가 넓어 세로 공간 부족)**: 12px gap 우선 보장 위해 캔버스 축소, 좌우 패딩 20px 초과 → 캔버스가 탑바/탭바보다 좁아 보일 수 있음.
- 내부 요소: 캔버스 크기 변화에 **비례하여 동일 비율 스케일**.

## Background Blur
- 배경 유무·종류(이미지/단색 등) 무관, **어떤 상태에서든 배경 전체를 덮는 Blur 도형을 별도 레이어로 항상 적용**.
- 설정값: Fill `Transparency/Black-25`, Effect `Background blur`, 방식 `Uniform`, 값 `4`.
- 배경 케이스 목업(물/콘크리트/검정) → 원본 소스 [[C-001-캔버스-정책-v0.1]] (`raw/assets/C-001-캔버스-background-case.png`).

# 그룹

[[협업-캔버스-앱]]의 협업 단위. 그룹마다 캔버스·멤버·닉네임을 가진다.

## 생성 / 참여
- **생성** (A-005): 그룹명 입력 → 초대 코드 자동 발급, 생성자가 그룹장 등록.
- **참여** (A-004): 초대 코드 입력 → 그룹 합류.
- 최대 인원 **12명** 고정. MVP에서 그룹명·인원수 추후 변경 불가.

## 닉네임
- 그룹 내 닉네임은 [[기능정의서-v4]]부터 가입 시 입력(A-004/A-005), 수정은 S-102.
- 캔버스 객체에 작성자 닉네임 표시(C-202), 탈퇴 유저는 "알 수 없음".
- 닉네임 정책(글자수/금칙어/중복)은 디자인단 설계 예정([[기능정의서-v5]]).

## 목록 노출 (G-001)
- **조회**: 한 번의 API 호출로 전체 조회, 페이지네이션 없음 ([[G-001-무한파르페-정책설계]]).
- **조회 필드**: 그룹ID·프리뷰 이미지(대표 [[토핑]])·이름·활동 시간.
- **화면 노출**: 그룹 이름(라벨 칩), 대표 토핑, 상대 활동 시간. 그룹ID는 조회하되 화면 미표시.
- 미노출: 인원/최대인원수, 참여 인원 ([[기능정의서-v5]] 기준).
- **정렬**: 가입순/활동순/이름순 투표 → **활동순** 다수 선택(자주 보는 그룹 상단). 최종 확정 문구는 아님.
- **레이아웃/새로고침/나가기 반영**: [[무한-파르페-그리드]] 참고 (2-1 스태거, Pull-to-Refresh 전체 교체, 나가기 시 패턴 재계산).

## 그룹장

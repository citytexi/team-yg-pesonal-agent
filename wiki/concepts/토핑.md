# 토핑

[[협업-캔버스-앱]]에서 캔버스에 배치하는 **누끼 딴 사진 객체**. 파르페(디저트) 위에 토핑을 얹는 메타포.

## 명칭 변천
- [[기능정의서-v2]]·[[기능정의서-v3]]: "누끼 사진" / "사진".
- [[기능정의서-v4]]부터 **"토핑"**으로 명칭 통일 (C-305 "토핑 편집", C-306 "토핑 테두리 편집").

## 라이프사이클
1. 촬영(C-101) 또는 갤러리(C-102)로 원본 사진 확보.
2. [[누끼-따기]](C-103)로 배경 제거 → 투명 PNG.
3. 수동 편집: 누끼 영역(C-104), 테두리·크기(C-105/C-306).
4. 캔버스 배치(C-106): 좌표(x, y, z-index), 크기. 확정 시 서버 저장 + 그룹 브로드캐스트.
5. 배치 후 수정: 파르페 편집 모드(C-301) → 토핑 편집(C-305, 자기 것만).

## 편집 정책
- MVP 편집 항목 = **크기, 테두리** 2가지로 고정.
- 자신이 올린 토핑만 위치/편집 가능.
- z-order·회전 허용 범위 미정([[open-questions]]).

## 반환 데이터
- 스티커(토핑) 아이디, 유저 아이디, 좌표(x, y, z).

## 대표 토핑 (G-001)
- 그룹 목록에서 각 그룹은 **대표(최신) 토핑** 프리뷰 1개로 표현됨 ([[무한-파르페-그리드]]).
- 프리뷰 이미지 크기는 디자인 고정값(가변 아님), 지연 로딩 + 캐싱.
- 클릭 시 개별 토핑을 잡고 있는 **C-001**로 이동.
- 우측 하단 라벨(이름 칩 + 상대시간), NEW 표시는 [[무한-파르페-그리드]] 라벨 정책 참고.

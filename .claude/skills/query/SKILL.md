# query — 위키 기반 질의·분석 워크플로우

`wiki/` 전체를 종합해 질문에 답하거나 분석을 만든다.
세부 규칙은 항상 `wiki/CLAUDE.md`(위키 스키마)를 따른다.

## 단계

1. **index.md 읽기**: 먼저 `wiki/index.md`를 읽어 관련 페이지를 파악한다.
2. **관련 페이지 읽기**: index에서 파악한 페이지들을 읽는다. 필요하면 `overview.md`, `open-questions.md`도 확인.
3. **출력 형식 선택**: 질문 성격에 맞게 고른다.
   - 일반 분석 → 마크다운 본문
   - 항목 비교 → 비교표 (마크다운 테이블)
   - 발표용 → Marp 슬라이드 (frontmatter에 `marp: true`)
   - 데이터 시각화 → matplotlib 코드 블록
4. **답변 작성**: 종합해 출처 링크(`[[페이지명]]`)를 포함한 답변을 제공한다.
5. **저장 여부 판단**: 재사용 가치가 있는 분석·비교 결과는 `wiki/synthesis/[주제].md`에 저장하고, `overview.md` 갱신 필요 여부를 판단한다.
6. **log.md 추가**: 파일 끝에 `## [YYYY-MM-DD] query | [질문 요약]` 한 줄 기록.

# 1. 토핑 구성

토핑 = **Img + Grouptag-Chip**, 3타입 (`1` / `2-Left` / `2-Right`)

---

# 2. 기본 규격

- 토핑 프레임: **180×180**
- Img: **144×144** (프레임 정중앙 기준)
- Chip: **회전 없음**
    - Timestamp 텍스트 컬러 = `프로필 이미지 및 토스트 닉네임 텍스트 컬러 규칙` 준수 (랜덤 X)
- Z: **Img > Grouptag-Chip** (항상)

---

# 3. 타입별 변형

정중앙 배치 상태(오프셋 0)를 기준으로 한 **타입별 고정 상수**.

## Type 1 (1열)

- Image Translate: X **-10**, Y **10**
- Image Rotate: **-6°**
- Chip Offset: **-9**

## Type 2 - Left (2열 좌)

- Image Translate: X **14**, Y **-6**
- Image Rotate: **-4°**
- Chip Offset: **-6**

## Type 2 - Right (2열 우)

- Image Translate: X **8**, Y **-12**
- Image Rotate: **8°**
- Chip Offset: **-12**

---

# 4. 반복 규칙

2-1-2-1 그리드 위에서 타입이 아래 순서로 반복된다.

```
        [ 1 ]
[ 2-Left ]  [ 2-Right ]
        [ 1 ]
[ 2-Left ]  [ 2-Right ]
        ......
```

---

# 5. 오버플로우 = 안 자름 (Clip Contents OFF / overflow: visible) ✅

- 144 Img를 회전·이동하면 코너가 180 프레임을 넘어감.
- 특히 극단적 케이스(누끼 거의 안 따진 풀블리드 사진)에서 심함.
- **이 삐져나온 부분을 자르지 않고 그대로 노출한다.**
- 삐져나온 픽셀은 인접 토핑을 밀지 않기 때문에, 자르지 않고 다 노출해서 자연스러운 분위기를 살린다.

> **정답**: 회전으로 프레임 밖으로 삐져나온 픽셀을 그대로 노출 (clip OFF).
> **오답**: 프레임(점선) 경계에서 잘라내는 것 — `이건 아님!`

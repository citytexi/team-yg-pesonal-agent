# 1. 측정 기준

- 간격은 **Topping (180×180) 프레임**을 기준으로 잰다.
- Img 기울기(회전) 때문에 프레임 밖으로 삐져나가는 부분이 생겨 헷갈릴 수 있으나, 갭은 회전된 Img가 아니라 **180 프레임** 기준으로 측정한다.
    - (참고: 삐져나온 픽셀 오버플로우 처리는 `G-001-토핑-정책-v0.1.md` 참고 — 자르지 않고 노출.)

---

# 2. 갭 규칙

세로 방향 위→아래로 반복되는 2-1-2-1 패턴 기준.

| 구간 | 방향 | 갭 |
| --- | --- | --- |
| 템플릿 체리 ↔ 첫번째 Type 1 | 세로 | 12px |
| Type 1 ↔ Type 2-Left | 세로 | 12px |
| Type 1 ↔ Type 2-Right | 세로 | 24px |
| Type 2-Left ↔ Type 2-Right | 가로 | 0px |
| Type 2-Right ↔ Type 1 | 세로 | 12px |

- 위 패턴이 그리드 전체에 걸쳐 **반복**된다.

## 2.1 끝 처리 (마지막 → 접시)

| 구간 | 방향 | 갭 |
| --- | --- | --- |

| 마지막 Type 2-Right ↔ 템플릿 접시 | 세로 | 0px |

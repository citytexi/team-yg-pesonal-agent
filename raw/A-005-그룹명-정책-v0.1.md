# 1. 생성 규칙

## 1.1 허용 문자
- **한글, 영문, 숫자, 공백(스페이스바)** 만 입력 가능.
- 특수문자·이모지 입력 불가.

## 1.2 글자 수 제한
- **최소 1자 ~ 최대 10자.**
- 글자 사이 공백(띄어쓰기)도 **1글자로 카운트**.

## 1.3 공백(띄어쓰기) 제한
- 맨 **앞·맨 뒤에는 공백을 넣을 수 없음**.
- 글자 사이 공백은 **1칸만 허용**, **연속 공백은 제한**.

---

# 2. 입력 예시 가이드

## 2.1 통과 케이스
- `열다섯글자열다섯글자` — 공백 없이 10자 충족.
- `홍 길동` — 글자 사이 공백 1칸이므로 가능.

## 2.2 불통과 케이스
- `   ` — 공백으로만 채운 그룹명 → 불가.
- ` 길동` 또는 `길동 ` — 시작/끝에 공백 포함 → 불가.
- `홍  길동` — 글자 사이 공백 2칸 연속 → 불가.
- `열 다 섯 글 자 열 다 섯 글 자` — 10자 초과 → 불가.

# 1. Button-Date (캘린더 일자 버튼)

일자 버튼의 4가지 상태.

| # | 상태 | 정의 |
| --- | --- | --- |
| 1 | **Default** | 2·3·4(Selected·Today·Disabled)가 아닌 모든 경우 |
| 2 | **Selected** | 사용자가 선택한 날짜 |
| 3 | **Today** | 오늘 날짜 |
| 4 | **Disabled** | 캔버스를 볼 수 없는 미래의 날짜 |

- **우선순위**: Selected(2)이자 Today(3)인 경우 → **Selected(2)로 노출**.

---

# 2. Chip-Indicator (토핑 인디케이터)

해당 날짜의 토핑 업로드 여부 표시.

| 값 | 정의 |
| --- | --- |
| **Upload/True** | 해당 날짜에 업로드된 토핑이 **1개 이상** |
| **Upload/False** | 해당 날짜에 업로드된 토핑이 **0개** |

---

# 3. List-Date (캘린더 일자 컴포넌트)

**Button-Date + Chip-Indicator** 로 구성되는 캘린더 셀 단위.

| 값 | 정의 |
| --- | --- |
| **Upload/True** | 해당 날짜에 업로드된 토핑이 **1개 이상** |
| **Upload/False** | 해당 날짜에 업로드된 토핑이 **0개** |

- **예외**: Button-Date가 **Disabled**인 경우 Chip-Indicator는 **항상 False** 처리.

## 상태 조합 (시각 케이스)

- Default & 토핑 O / Default & 토핑 X
- 선택한 날짜 & 토핑 O / 선택한 날짜 & 토핑 X
- 오늘 날짜 & 토핑 O / 오늘 날짜 & 토핑 X
- 미래의 날짜 (Disabled → 토핑 X 고정)

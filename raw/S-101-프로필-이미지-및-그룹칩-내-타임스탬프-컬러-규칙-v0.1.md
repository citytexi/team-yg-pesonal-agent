# 1. 프로필 이미지 정책

- 프로필 사진은 **닉네임 첫 글자가 들어간 원형 프로필 이미지(Nametag-Chip)** 로 노출.
- Nametag-Chip는 총 **6개 컬러**(연핑크, 진핑크, 체리, 그레이, 멜론, 푸딩)별로 **각 2개 타입** 존재 → **총 12종 중 1종 랜덤** 배정.
- (프로필 이미지 및 토스트 닉네임 텍스트 컬러 규칙과 동일. → `raw/S-101-프로필-닉네임-컬러-규칙-v0.3.md`)

---

# 2. 그룹칩 내 타임스탬프 컬러 매핑 정책

- G-001에서 그룹 대표 토핑과 함께 그룹 정보를 나타내는 **Grouptag-Chip**이 노출됨.
- Grouptag-Chip 속 **Timestamp 텍스트**는 해당 그룹에서 **마지막으로 변화를 가한**(토핑 추가, 토핑 편집, 배경 추가, 배경 변경) 유저의 **Nametag-Chip 타입에 따라 아래와 같이 강제 매핑**된다.

| 컬러 (타입) | 텍스트 컬러 |
| --- | --- |
| 연핑크 (Type 1, 2) | Cherry-100 |
| 진핑크 (Type 3, 4) | Cherry-200 |
| 체리 (Type 5, 6) | Cherry-300 |
| 그레이 (Type 7, 8) | White |
| 멜론 (Type 9, 10) | Melon |
| 푸딩 (Type 11, 12) | Pudding |

> 참고: Grouptag-Chip **Timestamp** 매핑(여기)과 토스트 **닉네임** 매핑(`S-101-프로필-닉네임-컬러-규칙-v0.3.md`)은 **서로 다른 UI 요소용 별개 매핑**이다. Cherry 계열은 단계가 다르고(Timestamp: 100/200/300 · 토스트 닉네임: 200/300/400), 그레이/멜론/푸딩은 동일. 같은 값이어야 하는 규칙이 아니므로 상충이 아니다.

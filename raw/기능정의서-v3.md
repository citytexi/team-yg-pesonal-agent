# 기능정의서_MVP v3

- **날짜**: 2026년 6월 7일
- **텍스트**: 3차 기능정의서
- **출처**: Notion 페이지 캡처 (2장), 원본 접근 불가(게스트)
- **v2 대비 주요 변경 (페이지 비고)**:
  - 사진 초기 배치 / 추후 수정 분기
  - 그룹명/인원수 수정 기능 삭제
  - 그룹장 기능 삭제

> 협업 캔버스 앱(그룹 단위, 매일 자동 생성되는 공유 캔버스에 누끼 사진을 배치)의 화면 단위 기능정의 3차. 화면 ID 접두사: `A-`(Auth), `G-`(Group), `C-`(Canvas), `S-`(Sidebar), `SY-`(System).

## v2 → v3 변경 요약

- **C-002 제거**: v2의 캔버스 편집 허브(C-002) 삭제. C-101/102는 이제 C-001에서 직접 진입, C-301(파르페 편집)도 C-001에서 진입.
- **C-301 개편**: v2 "캔버스 배경 변경" → v3 "파르페 편집 모드 진입"(배경 변경 + 누끼 사진 편집 통합 진입점).
- **C-305/C-306 신설**: 누끼 사진 편집(삭제/각도·크기/위치)·테두리 편집을 별도 화면으로 분리. v2 C-305("수정 완료")는 사라짐.
- **그룹장 기능 삭제**: S-103 예외처리의 "그룹장 권한 이양" 제거(비고에 "그룹 삭제 정책"만 남음). 페이지 비고에 "그룹장 기능 삭제" 명시.
- **그룹명/인원수 수정 기능 삭제**: 페이지 비고에 명시(A-005 비고와 일치).
- **사진 초기 배치 / 추후 수정 분기**: 최초 배치(C-101~106)와 이후 수정(C-301→C-305/306)을 경로상 분리.

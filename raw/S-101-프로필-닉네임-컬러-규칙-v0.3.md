# 1. 프로필 이미지 정책

- 프로필 사진은 **닉네임 첫 글자가 들어간 원형 프로필 이미지(Nametag-Chip)** 로 노출.
- Nametag-Chip는 총 **6개 컬러**(연핑크, 진핑크, 체리, 그레이, 멜론, 푸딩)별로 **각 2개 타입** 존재 → **총 12종**.
- 계정당 **12종 중 1종 랜덤** 배정.

---

# 2. Nametag-Chip 타입 고정 정책

- 타입은 **계정 생성 시 1회 랜덤 배정**, 이후에는 변경되지 않는다.
- **동일한 닉네임을 사용하는 사용자라도 계정이 다르면 서로 다른 타입**이 배정될 수 있다.
- 한 번 배정된 타입은 **모든 화면에서 동일하게 유지**되며, 앱 재접속·그룹 변경에도 변하지 않는다. 별도의 타입 변경 기능이 추가되지 않는 한, 계정이 유지되는 동안 동일 타입을 사용한다.

## 예시 (실명 마스킹 · 더미)

- 계정 A (닉네임 `홍씨`) → **Type 1** 배정 → 모든 화면에서 Type 1 유지
- 계정 B (닉네임 `홍씨`) → **Type 8** 배정 → 모든 화면에서 Type 8 유지
- 즉, **닉네임이 같아도 사용자(계정)가 다르면** 서로 다른 Nametag-Chip과 토스트 닉네임 컬러를 가질 수 있다.

# 3. 토스트 알림 닉네임 컬러 매핑 정책

- 공용 캔버스에서 액션 발생 시, 동접 중인 그룹원에게 토스트 알림이 뜬다.
- **토스트 속 닉네임 텍스트** 컬러는 유저의 **Nametag-Chip 타입에 따라 아래와 같이 강제 매핑**된다.

| 컬러 (타입) | 텍스트 컬러 |
| --- | --- |
| 연핑크 (Type 1, 2) | Cherry-200 |
| 진핑크 (Type 3, 4) | Cherry-300 |
| 체리 (Type 5, 6) | Cherry-400 |
| 그레이 (Type 7, 8) | White |
| 멜론 (Type 9, 10) | Melon |
| 푸딩 (Type 11, 12) | Pudding |

# Spec: 상단 바 (YGTopBar)

- 대상: `core:designsystem` — `component/ygtopbar/`
- 관련: [ADR-0010](../../adr/0010-custom-compositionlocal-theme.md)(자체 테마) · [design-system](../../architecture/design-system.md) · [ygiconbutton](2026-07-12-ygiconbutton.md) · [ygchipbutton](2026-07-16-ygchipbutton.md) · PR #152(`feature/#127-topbar`)

> 상태·날짜·대상·관련은 위 frontmatter가 단일 출처. 본문은 설계 내용에 집중.

## 목표
화면 상단 내비게이션 바. 좌측 아이콘 버튼 + 우측 타이틀/액션 슬롯 구성을 공유하는 **4개 프리셋 변형**을 노출한다. 공통 레이아웃은 private `YGTopBarContent`가 담당.

## 범위
- 포함: 4개 변형(뒤로가기/상세/빈/기본) + 공통 레이아웃, `YGIconButton`·`YGChipButton` 재사용, 토큰 매핑, 프리뷰.
- 제외:
  - 클릭 동작(내비게이션·메뉴 열기 등) — `onIconClick`/`onChipClick` 콜백만 노출.
  - 타이틀 문구 생성 — 호출자 주입.
  - 스크롤 연동·상태(collapsing 등) — 미지원(정적 바).

## API / 인터페이스
```kotlin
@Composable fun YGTopBarBack(onIconClick: () -> Unit, modifier: Modifier = Modifier)
@Composable fun YGTopBarDetail(title: String, onIconClick: () -> Unit, modifier: Modifier = Modifier)
@Composable fun YGTopBarEmpty(onIconClick: () -> Unit, modifier: Modifier = Modifier)
@Composable fun YGTopBarDefault(onIconClick: () -> Unit, onChipClick: () -> Unit, modifier: Modifier = Modifier)
```

### 변형 매핑
| 변형 | 좌측 아이콘 | 우측 슬롯 | 콜백 |
|------|-------------|-----------|------|
| `YGTopBarBack` | `ic_caret_left`("뒤로가기") | 없음 | `onIconClick` |
| `YGTopBarDetail` | `ic_caret_left`("뒤로가기") | `title` 텍스트(`body.b01R`, `Gray.Gray800`) | `onIconClick` |
| `YGTopBarEmpty` | `ic_hamburger`("메뉴") | 로고 이미지(현재 `ic_plus` placeholder) | `onIconClick` |
| `YGTopBarDefault` | `ic_hamburger`("메뉴") | 로고(placeholder) + `YGChipButton`("새 그룹", start `ic_plus`) | `onIconClick`, `onChipClick` |

## 동작 / 상태
- Stateless presentational. 각 변형은 공통 `YGTopBarContent`(private)를 서로 다른 아이콘/슬롯으로 구성.
- 좌측 아이콘: `YGIconButton(size = YGIconButtonSize.SIZE_44)`, `contentDescription` 지정(접근성).
- `YGTopBarDefault`의 칩: `YGChipButton(colors = YGChipButtonColorsDefaults.CherryBackgroundPressed)`.

## 레이아웃 / 토큰 매핑 (심볼명)
| 요소 | 토큰 / 값 |
|------|-----------|
| 루트 패딩 | top/bottom `padding3`, start `padding3`, end `padding7` (`YGTheme.layout.padding.*`) |
| 좌측 아이콘 | `YGIconButton`, `YGIconButtonSize.SIZE_44` |
| 타이틀 텍스트(Detail) | `YGTheme.typography.body.b01R`, `YGAtomicColors.Gray.Gray800` |
| 아이콘 리소스 | `R.drawable.ic_caret_left` / `ic_hamburger` / `ic_plus` |

## 파일 구성
- `component/ygtopbar/YGTopBar.kt` — public 변형 4종 + private `YGTopBarContent`.
- 프리뷰: `@Preview` + `YGCustomTheme`(흰 배경 위 4변형 세로 나열).

## 주의 / 열린 질문
- **로고 placeholder**: `YGTopBarEmpty`/`YGTopBarDefault`의 로고 자리는 `R.drawable.ic_plus` 임시 사용(코드 주석 `todo: parfait logo 로 변경 예정`). parfait 로고 에셋 확정 시 교체.
- **원자 색 직접 참조(과도기)**: 타이틀 색 `YGAtomicColors.Gray.Gray800` 직접 참조.
- **프리뷰 방식**: `@Preview`+`YGCustomTheme` 계열.

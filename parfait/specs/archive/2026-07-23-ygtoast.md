# Spec: YGToast 토스트 + 노출 정책 (YGToastHost)

- 관련: [ADR-0010](../../adr/0010-custom-compositionlocal-theme.md)(자체 테마) · [design-system](../../architecture/design-system.md) · 위키 [[Toast-공통-정책]]

## 목표
화면 상단에서 내려오는 토스트. **표시(YGToast + YGToastType)** 와 **노출 정책(YGToastPolicy/YGToastHost)** 을 분리.
화면은 `show(type)`만 하고, 다중 토스트 스택·자동 소멸·스와이프 닫기·애니메이션은 Host가 담당.
(YGAlert와 동일한 Policy/Host 패턴 — 단 YGToast는 **다중 스택**, YGAlert는 단일 슬롯. [ygalert](2026-07-23-ygalert.md) 참조.)

## 범위
- 포함: 타입별 토스트 컴포저블, 스택형 정책 홀더, Host(자동 소멸·위로 스와이프 닫기·슬라이드 인/아웃).
- 제외: 문자열 생성·다국어 매핑(호출자 소유). 타입에 완성 문자열/값만 담아 전달.

## API / 인터페이스
```kotlin
sealed interface YGToastType {
    data class InviteCode(val text: String) : YGToastType   // 초대코드 복사 등
    data class Edit(val text: String) : YGToastType         // 편집 안내
    data class Record(val userName: String, val time: String) : YGToastType  // "{userName}님이 {time} 전에 쌓았어요"
}

@Composable
fun YGToast(type: YGToastType, modifier: Modifier = Modifier)

data class YGToastItem(val id: String, val type: YGToastType, val visible: Boolean = true)

@Stable
class YGToastPolicy {
    val toasts: SnapshotStateList<YGToastItem>   // 다중 스택
    fun show(type: YGToastType)                  // add(0, …) — 나중 것이 위로
    internal fun setVisible(id: String, visible: Boolean)
    internal fun removeToast(id: String)
}

@Composable fun rememberYGToastPolicy(): YGToastPolicy
@Composable fun YGToastHost(policy: YGToastPolicy, modifier: Modifier)
```

## 레이아웃 / 토큰 매핑 (심볼명)
- 루트 `Box`(`CenterStart`) — `fillMaxWidth`, 배경 `YGAtomicColors.Transparency.Black75`, 패딩 세로 `YGTheme.layout.padding.padding6` / 가로 `padding4`.
- 타입별 텍스트(`when(type)`):
  - `InviteCode`: `body.b02SB` · `YGAtomicColors.Melon.Melon600`.
  - `Edit`: `body.b02SB` · `YGAtomicColors.Pudding.Pudding600`.
  - `Record`: `buildAnnotatedString` — `userName`은 `b02SB`/`Pudding.Pudding500`, `"님이 {time} 전에 쌓았어요"`는 `b02R`/`Gray.Gray100`.

## 노출 정책 (YGToastPolicy / YGToastHost)
- **다중 스택**: `toasts`(`mutableStateListOf`). `show()`가 `add(0, item)` — 최신이 리스트 선두(위키 "나중 것 위로 스택" 일치).
- **자동 소멸**: `YGToastHost`가 각 토스트 `key(id)` 안에서 `LaunchedEffect(id)` `delay(DISMISS_DELAY)` 후 `setVisible(false)` + `removeToast(id)`.
- **애니메이션**: `AnimatedVisibility` + `slideInVertically/slideOutVertically`(위에서 진입·이탈, `tween(ANIMATION_DURATION)`).
- **위로 스와이프 닫기**: `draggable`(Vertical) — 음수 델타만 누적, `onDragStopped`에서 오프셋 `SWIPE_DISMISS_THRESHOLD` 미만 또는 속도 `FLING_DISMISS_VELOCITY` 미만이면 `setVisible(false)`+`removeToast`, 아니면 리셋.
- 상수: `DISMISS_DELAY`(위키 정책의 "2초"에 대응) · `ANIMATION_DURATION` · `SWIPE_DISMISS_THRESHOLD` · `FLING_DISMISS_VELOCITY`(파일 private const, 값은 코드 SoT — 파르페 규율상 미기재).

## 파일 구성
- `component/ygtoast/YGToast.kt` — `YGToastType`(sealed) + public `YGToast` + `@Preview`.
- `component/ygtoast/YGToastPolicy.kt` — `YGToastItem`·`YGToastPolicy`·`rememberYGToastPolicy`·`YGToastHost`.

## 주의 / 열린 질문
